# Dearlog 나의 행성 & Nova 투명 PNG 에셋 v3

이번 패키지는 개발자에게 바로 전달하기 위한 **개별 투명 PNG 에셋 ZIP**입니다.  
소개 이미지/시트만 들어있는 파일이 아니라, 앱에서 각각 불러올 수 있도록 파일을 분리했습니다.

## 포함 내용

### Nova
- 기본 Nova: 1개
- 포즈: 3개
- 표정: 5개
- 안테나: 4개
- 가방: 4개
- 소품: 4개

### 행성
- 기존 예쁜 행성 에셋 유지
- 기본 행성, 고리, 별, 구름, 오브제 포함

## 참고

- 모든 개별 에셋 PNG는 투명 배경입니다.
- 원본 3D 모델 파일(.blend/.fbx)이 아니라 앱 적용/개발 전달용 PNG 에셋입니다.
- 최종 앱 구현 시 사이즈/좌표/애니메이션은 개발 쪽에서 조정하면 됩니다.
