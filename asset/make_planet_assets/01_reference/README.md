# Dearlog 나의 행성 & Nova 꾸미기 MVP 에셋

개발자 전달용 기본 에셋 패키지입니다.

구조:
- 일기별 행성 = 매일 감정 기록 결과
- 나의 행성 = 사용자의 프로필형 마음 공간
- Nova = 나의 행성에 사는 감정 기록 친구

구현 방식:
- MVP는 자유 드래그가 아닌 슬롯형/자동 배치 방식
- 각 아이템은 PNG 투명 배경 + metadata JSON의 defaultTransform 기준 적용
- 배치는 하단 탭 중 '마이' 탭에 '나의 행성' 카드 추가
- 커뮤니티 프로필 클릭 시 '행성 놀러가기' 진입
