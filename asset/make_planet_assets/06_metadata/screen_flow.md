# Dearlog 나의 행성 & Nova 화면 흐름

## 1. 마이 탭
- 상단에 `나의 행성` 카드 추가
- 카드 요소: 행성 미리보기, Nova, 최근 감정 요약, 방문자 수, 받은 별조각 수
- CTA: `내 행성 보기`

## 2. 나의 행성 화면
- 중앙: 메인 행성 + Nova
- 주변: 장착한 행성 아이템/위젯
- 버튼: `행성 꾸미기`, `Nova 꾸미기`, `공개 설정`, `공유하기`

## 3. 행성 꾸미기 화면
- 카테고리: 배경, 고리, 별조각, 위젯
- MVP: 아이템 선택 시 defaultTransform 위치로 자동 적용
- 저장 시 equippedPlanetItems 업데이트

## 4. Nova 꾸미기 화면
- 카테고리: 안테나, 가방, 손 소품, 표정, 옷
- MVP: 슬롯형 장착 방식
- 저장 시 equippedNovaItems 업데이트

## 5. 커뮤니티 연동
- 커뮤니티 프로필 클릭 → `행성 놀러가기`
- visibility가 public 또는 anonymous일 때만 방문 가능
- 방문자는 공감/별조각 남기기 가능
